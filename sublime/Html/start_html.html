快捷键

操作 作用
<html 生成Html固定格式
ctrl + s 保存
ctrl + shift + / 注释 或 取消上一次注释
ctrl + shift + d 复制上一行到下一行
ctrl + shift + h 格式化HTML5（已安装插件html-css-js prettify + node.js）
ctrl + x 删除当前行 或 剪切
ctrl + z 取消上一次编辑
ctrl + f 查询（单个文件内）
ctrl + d 多选，同时操作
ctrl + p 在当前目录中查找文件

偶尔但重要的标签

标签 作用
xmp / textarea 插入不被编辑的代码块
table + tr + td 简易表格
frameset(cols) + frame 分帧+联动
